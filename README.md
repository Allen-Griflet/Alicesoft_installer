#### 使用方法<br/>
１. README記載の対象ゲームのディレクトリ名からリンクを踏んでダウンロードしてください。<br/>
２. ダウンロード後、zipファイルを解凍(展開)してください。(Download/対象ファイル/すべてを展開/)<br/>
３. 解凍(展開)後、対象のゲームのフォルダへ解凍(展開)したファイルをコピーしてください。<br/>
４. Setup.exeを起動してください。<br/>
